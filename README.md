# Yashvardhan Patel — Resume Website

This repository contains the **HTML/CSS version of my resume**, designed to be clean, responsive, and easily exportable to PDF.

The project replicates a traditional resume layout while also being usable as a **web portfolio resume**.

---

## Features

* Clean and minimal resume layout
* Centered header with contact information
* Icons for phone, email, LinkedIn, and GitHub
* Structured sections for education, experience, projects, and skills
* Printable layout for PDF export
* Lightweight HTML + CSS implementation

---

## Tech Stack

* HTML5
* CSS3
* Font Awesome (for icons)

## Running Locally

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/resume-website.git
```

2. Open the project folder

```
cd resume-website
```

3. Open the file

```
index.html
```

in your browser.

---

## Exporting to PDF

You can export the resume to PDF using your browser:

**Chrome / Edge**

```
Ctrl + P → Save as PDF
```

Recommended settings:

* Paper Size: A4
* Margins: Default
* Background Graphics: Enabled
